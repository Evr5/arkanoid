Game Guide

Learn the rules, controls, and how to create your own levels!

Game Rules

The goal of the game is to destroy all the bricks on the screen by bouncing the ball off the racket and hitting them.

  • Each level contains bricks of various colors that award different points.
  • Some bricks may contain bonuses that help you progress through the game.
  • You start with 3 lives, and the game ends when you lose all your lives.

Game Controls

  • r: Reset the best score and current score to 0
  • ←: Move to the previous level
  • →: Move to the next level
  • a: Replay the current level
  • q: Quit the game
  • spacebar: Shoot a lazer when available
  • spacebar: Release the ball when it is stuck to the racket

Create Your Own Levels

You can create custom levels by adding text files to the ressources/levels directory.

  1. Save the file in .txt format with a numeric name (e.g., 1.txt, 2.txt).
  2. Each line in the file represents a brick, using the format:
X-coordinate Y-coordinate Color [Bonus]

Available colors:

  • defaultBrick
  • white
  • orange
  • cyan
  • green
  • red
  • blue
  • magenta
  • yellow
  • silver
  • gold

Available bonuses:

  • None
  • SlowDown
  • ExtraLife
  • WideRacket
  • StickyRacket
  • SplitBall
  • Lazer
